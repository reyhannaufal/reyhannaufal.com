## 5 tips to set up an accurate deadline

**1. Make a list of all the things you need to do**

By making a list of all the things you need to do, you can make sure that you have a clear idea of what you need to do. This will help you to prioritize your tasks. You can also use this list to make sure that you don't forget anything.

**2. Make it a habit**

By making it a habit, you can make sure that you don't forget to do it. You can also use this habit to remind yourself to do it.

**3. Set a time frame**

You can set a time frame for each task. By doing this each task will have a deadline. This will help you to make sure that you don't forget to do it.

**4. Focus on the task**

When you focus on the task, it will be easier to complete it. By doing focus on one task at a time, your productivity will increase and you will be more efficient.

**5. Retrospective and Planning**

When you complete a task, you can reflect on it. You can also use this to make sure that you are doing the right thing
