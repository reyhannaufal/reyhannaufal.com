## Run Locally

Clone the project

```bash
  git clone https://github.com/reyhannaufal/reyhannaufal.com.git
```

Go to the project directory

```bash
  cd reyhannaufal.com
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
